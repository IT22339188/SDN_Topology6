Communication Policy

Source Destination Status Reason
H1-H4 H5-H6 ✓ ALLOW Clients → Servers

Type Bash Script

Rule Installation Process

  1. 1
    Clear all existing flows on all 7 switches
  2. 2
    Install ARP rules for switch-to-switch communication
  3. 3
    Install ALLOW rules for H1-H4 → H5-H6 traffic
  4. 4
    Install ALLOW rules for H5-H6 → H1-H2 traffic
  5. 5
    Install DROP rules to block H1-H4 → H7-H8
  6. 6
    Install DROP rules to block H7-H8 from outside
  7. 7
    Install ALLOW rules for isolated H7 ↔ H8
  8. 8
    Install default DROP rule on each switch

ICMP Type Trick (S6)

Special Feature: Asymmetric communication using ICMP type filtering

ICMP Type Filtering at S6
ICMP Type 8 (Echo Request): H5 → H3 = BLOCKED (priority 200) ICMP Type 0 (Echo Reply): H5 ← H3 = ALLOWED (priority 300) Result: H3 can reply to H5, but H5 cannot initiate to H3!
